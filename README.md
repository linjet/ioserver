# Служба транспорта данных с распределением нагрузки на конечные точки
В принципе, это просто веб-сервер, но с глубокой кастомизацией, позволяющей не только добавлять свои собственные библиотеки обработки REST-запросов, но и определять полностью кастомную аутентификацию, в плоть до собственного механизма разборки данных из входящего потока.
Пока это только обработка TCP. В дальнейшем добавлю обработку UDP-трафика.
